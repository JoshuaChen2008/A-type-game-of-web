# 基于React+TypeScript+TailwindCSS的打字网站

一个用Vite手脚架搭建的React+TypeScript的前端项目，用来练习React+ts

项目支持随机单词生成、倒计时、输入错误统计、准确率计算、暗色模式切换、重启以及简单动画效果

- 随机生成英文单词
- 一键重新开始
- 支持亮色 / 暗色模式切换
- 使用 Framer Motion 实现基础动画效果



## 环境要求

- Node.js 版本：建议 18 或以上
- npm 版本：建议 9 或以上

## 可以通过以下命令检查版本

```
node -v
npm -v
```

## 安装依赖

克隆项目以后，在项目根目录执行:

```bash
npm install
```

## 本地开发：

启动后端口默认为5173/5174

```
npm run dev
```

## 构建项目:

```
npm run build

#预览构建结果
npm run preview

#代码检查
npm run lint
```
